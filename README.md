# Requirements 
Better mobile 
NETHUNTER KeX
Termux (playstore or fdroid)

# Setup
Download the Delta.sh file the make a folder in Dowload folder named (Delta) and copy the Delta.sh file in the Delta folder
run the commands given below
# Commands for Termux

cd ~/storage/downloads/delta

mv delta.sh $PREFIX/bin/delta

chmod +x $PREFIX/bin/delta

delta
